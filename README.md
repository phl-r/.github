# PHL-R: The Philippine R Registry 🇵🇭

### Building the open-source infrastructure for an archipelagic nation.

The **PHL-R** organization serves as the central hub for R software development in the Philippines. We consolidate specialized tools for spatial modeling, national statistics, and public health to ensure that Filipino researchers have access to world-class, locally-relevant software.

- **Primary Registry:** [phl-r.r-universe.dev](https://phl-r.r-universe.dev)
